# 项目说明

## 环境要求

- node@16+
- pnpm@6+, 通过 npm 安装 `sudo npm install -g pnpm`

## 开始前

- 使用 VS Code 进行开发，安装 `volar`(Vue Language Features) 插件
- `Vetur` 插件主要用于 Vue 2 项目，在 Vue 3 工作区内需将其禁用

## 开始

```bash
# 克隆项目
git clone <projectUrl>

# 进入项目根目录
cd <projectName>

# 安装依赖
pnpm install

# 运行项目, 开发阶段使用
pnpm dev

# 打包项目, 生产环境使用
pnpm build
```
